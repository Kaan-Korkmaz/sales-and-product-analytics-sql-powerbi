# Satış Analizi Projesi (SQL)

Bu proje, satış verileri üzerinde SQL kullanılarak yapılan uçtan uca bir
veri analizi çalışmasını içermektedir.\
Amaç, ham işlem verilerinin analiz edilerek iş kararlarını destekleyecek
anlamlı çıktılara dönüştürülmesidir.

------------------------------------------------------------------------

## Proje Kapsamı

Projede aşağıdaki analiz adımları yer almaktadır:

-   Veri profilleme ve veri temizleme
-   Satış performans analizleri
-   Müşteri ve ürün bazlı analizler
-   Zaman serisi ve trend analizleri
-   Segmentasyon çalışmaları
-   Raporlama amaçlı SQL view oluşturulması

------------------------------------------------------------------------

## Kullanılan Yapı

Analizler; bir satış fact tablosu ve müşteri ile ürün boyut tabloları
üzerinden gerçekleştirilmiştir.\
Tüm metrikler ve iş kuralları SQL içerisinde tanımlanmıştır.

------------------------------------------------------------------------

## Raporlama

Projenin sonunda oluşturulan SQL view'lar, Power BI veya benzeri BI
araçlarına doğrudan bağlanabilecek şekilde tasarlanmıştır.
